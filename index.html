NURBS.jl

NURBS.jl (or Non Uniform Rational B-Spline Surfaces) is a Julia library to build parametrical curves and parametrical surface based on the interpolation of polynomial bases. This library is developed by:

  • Elia Onofri - Ln (elia.onofri4@gmail.com)

  • Gianmarco Caramitti (g.caramitti@gmail.com)

  • Paolo Macciacchera (polmacra@outlook.it)

  • Giuseppe Santorelli (giu.santorelli15@gmail.com)

and would be maintained by the Computational Visual Design Laboratory (CVDLAB) of Università degli Studi di Roma Tre.

Dependencies

NURBS.jl has no dependencies but is build to work with:

  • Plasm

Docstrings conventions

  • Bold is used to point out theory concepts.

  • Monospace is used for everything code related.
